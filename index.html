Facebook HTML Uncomment

Helps Facebook App Developers debugging their application by notifying them when Facebook has commented the application server's output. It then shows the developer a icon to click on to see the server output Shows a small icon in the status bar which turns blue when a Facebook application has the error output hidden in a HTML comment. The icon then turns blue, allowing the developer to click the icon and see the original server output in a new window.

Install

Add-on can be loaded from https://addons.mozilla.org/en-US/firefox/addon/10139
